# Pre-Release Compliance Checklist

Status: Active
Purpose: Give the founder a simple, repeatable pre-release compliance review.

## Before Public Beta

- Privacy policy drafted and reviewed against actual app behavior
- Consent copy in onboarding/settings matches the real permissions and data flow
- AI remains disabled by default unless its legal position is explicitly cleared
- Delete-all-data flow behaves exactly as described in the current locked docs
- No hidden retention behavior exists

## Before Public Launch

- Privacy policy published and linked wherever required
- Play Data Safety answers prepared from real app behavior
- Financial Features declaration prepared if Play channel is used
- Terms of use drafted in plain language
- AI cross-border/SPDI gate explicitly resolved or AI remains off
- Retention vs delete-all-data conflict explicitly resolved

## Release Rule

If any item here is false, release is not ready.
