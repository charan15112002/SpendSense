# Test Runs Folder

Purpose: Store exported evidence bundles and run notes from founder and beta testing.

Recommended structure:

- one folder per test run
- folder name includes date, build ID, and tester alias

Example:

- `2026-03-29_SS-M2-founder-local-playstore-20260329-01_charan/`

Rule:

Every stored run must be traceable to one build ID and one flavor.
